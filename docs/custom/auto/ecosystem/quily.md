# Quily

**Part of the official Quilibrium ecosystem.** Relation to the network: Plugs into Quilibrium (an app or tool that connects to Quilibrium but runs on its own servers).

**An AI chatbot that helps you quickly find information about Quilibrium.** Ask about node operation, tokenomics, technical details, or anything else, and Quily searches through official and community sources to put together an answer.

Under the hood Quily uses **Retrieval-Augmented Generation (RAG)**: instead of relying on what the model happened to memorize during training, it looks things up in a curated set of documents first and then composes a response grounded in what it found. That makes the answers easier to trust and easier to fact-check, since they come from sources you can go and read. The generation step runs on **open source large language models**, so no part of the stack is a black box.

**Data sources:**
- The official Quilibrium documentation.
- Transcriptions from official Quilibrium live streams.
- Custom and community-contributed documents: Discord explanations, technical references, community guides. All of these live in the project repository, so anyone can review what Quily is reading from or propose additions.

You can use Quily on its **own web app** or directly inside the **Quilibrium community Discord server**, where it answers questions inline so you can ask without leaving the conversation.

Quily is **open source** and built by the Quilibrium community.
