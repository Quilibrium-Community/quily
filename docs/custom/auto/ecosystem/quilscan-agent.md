# QuilScan Agent

**Part of the official Quilibrium ecosystem.** Relation to the network: Plugs into Quilibrium (an app or tool that connects to Quilibrium but runs on its own servers).

**Manage your Quilibrium node from your browser**, instead of logging into the server through the terminal every time.

You install a small program (the agent) on the server where your node runs. It connects to QuilScan, and from there you get a **Node Console**: a web page that shows what your node is doing and lets you control it with buttons.

From the Console you can set up a brand-new node on an empty server, or take over an existing one you already have running. After that you can start it, stop it, restart it, update it, clean up old files, and update the agent itself, all from the browser.

The Console also shows you the things you usually check by hand: which version of the node software is running, how many peers it is connected to, how full the disk is, how much memory it is using, the block height, and the live log. It works on **Linux** servers and on **Mac (Apple Silicon)**.

Open source on GitHub. Built and maintained by the same team behind QuilScan and the QuilScan Wallet.

**A note on trust.** Once your server is connected to QuilScan, the QuilScan team can send commands to it. That is the whole point of the tool, but it does mean you are trusting them not to misuse that access (for example, by stopping or removing your node). **If you are storing anything important on the same server, or running a large amount of `$QUIL`, factor that in.**

The [code is open source](https://github.com/quilscan-com/quilscan-agent), so if you want a closer look before installing, point your favourite AI agent at the repo and have it audit the version you're about to run.
