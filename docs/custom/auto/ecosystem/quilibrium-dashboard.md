# Quilibrium Dashboard

**Part of the official Quilibrium ecosystem.** Relation to the network: Built on Quilibrium (an app that runs on Quilibrium and needs the network to work).

**The official Quilibrium Inc. dashboard for developers and node operators who want to dig into how the network is behaving.** A useful thing to understand up front: **Quilibrium is fully encrypted by default**, which means an Ethereum-style block explorer (look up a transaction, see who paid what to whom) is not just absent here, it is not possible by design. Transactions are private. What you can observe is the network's behaviour at the protocol level, and that's what this dashboard exposes.

A non-exhaustive sense of what you can look at:

**Frame analytics.** Live data on frame production.

**Shard coverage.** Which shards are covered, which are at risk, and how the shard-out is progressing.

**Consensus and gossip.** A live feed of raw gossip messages traversing the network.

**Peer directory.** A view of the peers participating in the network.

It's deliberately low-level today, with a heavy focus on raw protocol data over summarised views.
