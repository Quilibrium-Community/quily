# Community Recap — #general

**July 3, 2026** | 12:33 AM UTC – 8:36 AM UTC

## Node Issues and Network Status

The network is currently stuck at frame 676591 with `archive_peers: 0 permanent` and `connect_mtls` failures. Multiple users confirmed the entire network is halted. Cassie indicated a patch (v2.1.0.25) is in testing to resolve this. She later noted that the migration has been completed.

## New User Questions and Project Clarifications

A new user, andrew29, asked whether mining is currently active and whether rewards exist. Mr.Franc explained that mining has been paused for about 1.5 years as the network prepares for a major upgrade. Mining historically yielded tokens (QUIL), and an ERC-20 bridge existed for token transfers but is now closed, expected to reopen after the launch. Cassie clarified that mining is not presently occurring, though nodes remain active for routing (e.g., Quorum functionality). She addressed the ERC-20 bridge's original purpose: to protect early miners from bad actors offering to "maintain" their nodes.

andrew29 expressed skepticism about full decentralization and questioned the project's viability. Several community members defended the project's vision, with Finn emphasizing the breadth of live features (Quorum, QNS, Q Storage) and Cassie describing Quorum as an "everything app" like WeChat but privacy-focused. Cassie also responded to a concern about child safety by noting zero-knowledge proofs for proof of legal age.

## Development Updates from Cassie

Cassie reported reconciling pull requests on the repository, ensuring alignment with already-locally-fixed code. She also identified the root cause of a bandwidth usage bug (reported via Telegram), which should result in a much quieter network once fixed. She described these PRs as critical fixes for the upcoming .25 release.
