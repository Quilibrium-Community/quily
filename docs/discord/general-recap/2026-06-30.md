## Community Moderation

TrustX announced plans to invite core community moderators to the GitHub repository, initially providing access to smart contract code for transparency review. He explained this phased approach is for security during early stages, with full open-source release planned once the code is thoroughly tested and stable. This announcement was met with skepticism from Otter, which TrustX addressed by noting healthy skepticism is welcome and the project will be judged by its code and product.
