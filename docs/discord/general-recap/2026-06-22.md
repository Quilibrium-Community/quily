# Community Recap — #general

**June 22, 2026** | 1:33 AM UTC – 11:24 PM UTC

## From Cassie

Cassie spent the day heads-down working through archive issues, restoring request processing flow. Shortly after, **shardframes began landing** — a major milestone as this is the first step to miners earning. However, the influx of shardframes revealed that global processing was single-threaded, causing a temporary halt. The fix is **batch verification of BLS signatures**, a technique already in the codebase that just needs to be plugged in at the right place. No patch update is required; only archive nodes are affected. Cassie noted this is a rare case where using the math behind cryptography makes something more complex run faster.

Later, she observed that a **large number of duplicate join requests** appeared across frames. This is not a `.24` bug but likely related to batching/reattempt logic she added to prevent message loss when an archive peer fails. It’s technically harmless but noisy.

## Network Progress and Observations

Community members reported seeing many duplicate requests. Users speculated that evictions may have begun. Cassie confirmed that node prover sets changing in bulk would be visible on dashboards or Quilscan, and that individual nodes would go “berserk” making leave decisions. Frame times improved, with ~50% of frames landing in 10–12 seconds. Requests per frame appeared capped at 100.

## Community Announcements

**QGame** launched the **Soonnet Runner**, a browser game where players squash bugs, clear FUD, type code to earn points, and progress from version 2.1.0 to 2.1.0.24. Some users joked about potential airdrops, but no official linkage was made.

## Moderation and Scams

Discussion arose about spam bots posting links and then deleting their own messages to avoid bans. Tyger (a moderator) explained they can only manually ban, and bot self-deletion makes tracking difficult. Users suggested screenshotting usernames. The community noted that effective moderation helps, but it’s a whack-a-mole problem.

## Off-Topic Discussion

A brief, nuanced conversation about immigration and refugee policy unfolded, with users sharing perspectives on systemic issues, empathy, and the challenge of balancing aid with domestic priorities. The discussion remained civil and underscored the broader societal context that makes Quilibrium’s mission feel timely to some.
