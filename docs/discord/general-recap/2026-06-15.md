# Community Recap — #general

**June 15, 2026** | 12:00 AM UTC – 11:34 AM UTC

## Wallet Security Discussion
Trystan and Mr.Franc discussed the risks of hot wallets versus hardware wallets like Ledger. Mr.Franc warned that clicking malicious links can steal funds, and recommended using a Ledger. Trystan noted they would order one due to upcoming regulations.

## From Cassie
Cassie [LEAD DEV] reported merging Daz's PR that threads transactions through to sync. She is currently working on closing out the "wiped stores, allocations present, some leaving, handle accordingly" scenario reported by Black Swan.

## Quilscan Agent Migration
Quilscan announced a service migration that may cause some nodes to temporarily lose their agent connection. They provided instructions to restart the agent service (Linux: `sudo systemctl restart quilscan-agent`, macOS: `sudo launchctl kickstart -k system/com.quilscan.agent`) and wait 30–60 seconds before refreshing the dashboard.

## Network Progress: Evictions and Shard Coverage
alcohol asked whether the previous shard coverage is still valid or if nodes must upgrade to version .24 and re-cover halt-risk shards to trigger eviction. LaMat clarified that evictions are the missing piece: fixing the eviction code so offline nodes can be evicted is needed. After that, the network would likely halt again and re-shard with fully live nodes.
