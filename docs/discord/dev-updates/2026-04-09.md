**Updates for Quorum Development**
March 25 – April 9, 2026

This list is for the web app, these changes are not yet live but you can test them on https://test.quorummessenger.com/

🎉 **New Features**

💬 Messages & Media
• Send YouTube links and get a clean preview thumbnail — no external tracking, the image is embedded in the message itself
• Send text and an image together as a single message instead of two separate ones

👥 New Member Badge
• New space members get a 🌱 seedling badge next to their name for 7 days, so the community knows who just joined

⚙️ Device Naming
• Name each of your devices so you can tell them apart in settings (e.g. "Work Laptop", "Phone")

🎨 Panels Redesign
• Threads, Bookmarks, Notifications, Pinned Messages, and Search panels all have a refreshed card style that's cleaner and easier to scan

✏️ Drag & Drop Uploads
• Dragging a file over any upload area (avatar, banner, emojis, stickers) now gives clear visual feedback so you know exactly where to drop it
