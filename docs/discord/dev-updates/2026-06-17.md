# dev-updates - June 17, 2026

## Cassie — 11:07 AM UTC

While we are awaiting tester feedback and current builds' signatures (assuming testing feedback is copacetic, there is a report but needs confirmation), klearu has been updated with new model support, enhanced specifically for klearu: transformer-driven image generation, with support for specialized text rendering.
