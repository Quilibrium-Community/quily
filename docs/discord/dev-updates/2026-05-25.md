# dev-updates - May 25, 2026

## LaMat — 4:25 PM UTC

**Updates for Quorum Development**
April 16 – May 25, 2026

This list is for the web app, these changes are already live on https://app.quorummessenger.com/

🎉 **New Features**

✏️ Typing Indicators
• See who's typing in real time, in both space channels and direct messages
• Multiple people typing at once are all shown

👥 User Profile in DMs
• Open a sidebar in any DM to see the other person's full profile
• Add a private note to any contact, only visible to you, so you can remember who someone is

💬 Floating Composer in Channels
• The message composer now floats as an overlay so messages flow flush above it instead of being cut off behind it

🔍 Search Speed
• Search no longer blocks app startup
• First search in a space or DM is near-instant
• After a restart, search results come back in roughly 10ms instead of rebuilding from scratch

⚡ Message Quick Actions
• Hold Shift while hovering a message to swap the reactions row for quick action icons: copy link, copy text, edit, pin, bookmark, delete
• Shift+click skips the confirmation prompts for delete and pin, so power users move faster

🎨 Image Download
• Download button added to the image modal
