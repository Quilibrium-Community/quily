# announcements - May 29, 2026

## LaMat — 7:23 PM UTC

@everyone

⚠️ Heads up — crypto phishing scam targeting our community (and others) on GitHub

  A fake GitHub repo ("valvepsro466/Support") is scraping GitHub issues, and @mentioning the original authors so you get a GitHub notification. An automated bot reply then
  links to a fake "GitHub official support" site (github-officialweb3.pages.dev) and a Gmail address. The "live support chat" is a wallet drainer that asks for your seed phrase.

  Do NOT:
  • click any "support" / "live agent" links inside GitHub issues
  • connect your wallet or enter your seed phrase on any such page

  GitHub support never DMs you a third-party chat link, and no legitimate support ever asks for a seed phrase. If you got @mentioned in one of these issues,
  ignore it and unsubscribe from the thread. 

We've reported it to GitHub and Cloudflare.
